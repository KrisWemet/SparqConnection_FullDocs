# Data Model for Sparq Connection

## Core Tables:
- **Users**  
- **Couples**  
- **DailyPrompts**  
- **PromptResponses**  
- **QuizResponses**  
- **ForumPosts & Comments**  

## New Tables:
- **MoodLogs**  
- **Challenges**  
- **HealthScores**  

## Relationships:
- **Users ↔ Couples**  
- **Daily Prompts ↔ Prompt Responses**  
- **Quizzes ↔ Quiz Responses**  
- **Couples ↔ Challenges**  
- **Couples ↔ HealthScores**
